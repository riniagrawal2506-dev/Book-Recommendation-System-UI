# Book Recommendation System Using Machine Learning

## Research Paper Title
Book Recommendation System Using Machine Learning

## Domain
Machine Learning / Recommendation Systems

## Overview
This project replicates the output interface presented in the selected research paper. The system displays book recommendations using cosine similarity and collaborative filtering techniques.

## Implementation
The frontend interface was developed using HTML and CSS.

## Pages Included
- Home Page
- Popular Books Page
- Recommendation Page
- Book Catalog Page

Screenshots of all developed pages are included in this repository.
